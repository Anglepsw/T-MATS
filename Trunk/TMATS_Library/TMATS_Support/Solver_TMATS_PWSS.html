T-MATS: 4D Piecewise State Space Model Library Block

This block represents a 4D state-space model.
This model will be of the form:

$$ (\dot{X} - \dot{X}_0) = A*(X - X_0) + B*(U - U_0)$$ $$(Y - Y_0) = C*(X - X_0) + D*(U - U_0)$$

Where X is the system state, U is the system input (assumed to be 1x1), and Y is the plant output. 0 values are the trim point values.

The 4 dimensions are represented as up to 3 "Environmental" parameters and 1 control parameter that may or may not be U. Note, if NumEnv_M is set to a value less than 3, tables must still be defined as 4D and unused Env values will be set to 0.
