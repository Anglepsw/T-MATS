T-MATS: MATLAB functions

The T-MATS package contains MATLAB functions for doing certain Thermodynamic operations. Information about these sets of function are located in section 4.5 of the users manual.

As a quick reference Table 9 is located below. They can be accessed by using the class structure TMATS.* .

TMATS.* functions
Class MethodDescription
Object = FlowDef()Flow object constructor
Thermodyanmic Table lookups:
[T] = h2t(h)Look up temperature[T] based on enthalpy[h].
[h] = t2h(T)Look up enthalpy[h] based on temperature[T]
[s] = PT2s(T,P)Look up entropy[s] based on pressure[P] and temperature[T]
[T] = sP2T(s,P)Look up temperature[T] based on entropy[s] and pressure[P]
