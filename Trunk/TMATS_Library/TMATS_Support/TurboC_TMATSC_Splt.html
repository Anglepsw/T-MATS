T-MATS: Splitter, Cantera Library Block
This block simulates the performance of a splitter/branch using basic flow fluid dynamic equations. To utilize this block a Cantera object must exist in the workspace as a global variable named fs, which may be created with the Cantera importPhase command. The fs variable must include all Species defined in the file TMATSC_canteraload.m.

Splitter Inputs:
Splitter InputDescription
Flow InFlow Data Input, 25x1 vector consisting of:
-1- W - gas path flow [pps]
-2- Tt - total temperature [degR]
-3- Pt - total pressure [psia]
-4- ht - total enthlapy [BTU/lbm]
-5to10- species array values, defined in TMATSC_canteraload.m
-11- s - entropy [BTU/(lbm*degR)]
-12- rhot - total density [lbm/ft3]
-13- Ts - static temperature [degR]
-14- Ps - static pressure [psia]
-15- hs - static enthalpy [BTU/lbm]
-16- rhos - static density [lbm/ft3]
-17- Vflow - flow velocity [ft/sec]
-18- MNin - Mach number
-19- A - flow area [in2]
-20- gamt - total specific heat ratio
-21- gams - static specific heat ratio
-22to25- future use
BPRBranch Pressure Ratio

Splitter Outputs:
Splitter OutputDescription
Flow SecondaryFlow Secondary, 25x1 vector defined as Flow In above
Flow PrimaryFlow Primary, 25x1 vector defined as Flow In above
S_DataSplitter Data, additional calculation data
