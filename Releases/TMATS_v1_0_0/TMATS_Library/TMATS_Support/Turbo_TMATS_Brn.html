T-MATS: Burner Library Block
This block simulates the performance of a burner using basic thermodynamic equations and properties.

Burner Inputs:
Burner InputDescription
FuelFlwInFuel Flow Input [pps]
GasPthCharInGas Path Characteristics Input, 5x1 vector consisting of:
- W - gas path flow [pps]
- ht - enthalpy [BTU/lbm]
- Tt - Total Temperature [degR]
- Pt - Total Pressure [psia]
- FARc - Combusted Fuel to Air Ratio [frac]

Burner Outputs:
Burner OutputDescription
GasPthCharOutGas Path Characteristics Output, 5x1 vector consisting of:
- W - gas path flow [pps]
- ht - enthalpy [BTU/lbm]
- FARc - Combusted Fuel to Air Ratio [frac]
- Pt - Total Pressure [psia]
- Tt - Total Temperature [degR]
B_DataBurner internal calculation data

Burner Mask Variables:
Burner Mask VariableDescription
LHV_MFuel Lower Heating Value[BTU/lb]
dPnormBurner_MNormalized Burner deltaP accross the burner[1 - psi/psi]
Eff_MBurner Efficiency[frac]
